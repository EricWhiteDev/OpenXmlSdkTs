As a first step in the openFromZip method, open the content types part (named '[Content_Types].xml')
Read as an XDocument, and set the ctXDoc property of the OpenXmlPackage.
Modify the getContentType function to take the ctXDoc XDocument as an argument.
Modify getContentType to use LtXmlTs to get the content type from xtXDoc.  The XML looks like this:

        ContentType="application/vnd.openxmlformats-officedocument.wordprocessingml.document.main+xml" />
